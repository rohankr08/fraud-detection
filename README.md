# Fraud Detection System

This project is an AI-based fraud detection system that analyzes transaction data and detects suspicious activities.

## Features
- Detect fraudulent transactions
- Machine learning model integration
- Web interface using Flask

## Tech Stack
- Python
- Flask
- Machine Learning
- HTML/CSS

## How to Run
1. Clone the repo
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python app.py
